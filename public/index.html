Pure Grid System

Similar to other frameworks like bootstrap, Pure includes a grid system to help developers get the perfect page layout. The Pure grid system is based on fractions instead of the number 12. It supports fractions with denominators of 2,3,4,5,6,8 and 24 for extremely specific separations. To use a grid, you must start with a div with class .pure-g (pure grid). Then for its children the grid layout can be specified with classes of .pure-u-* where * could be a fraction like 1-2 or 22-24.

Responsive Grids

Pure's Grid system is for mobile devices by default, but they can also become responsive through an extended css file called grids-responsive.css. This file includes more classes for letting grids respond to the device size. Pure breaks it down into categories of small (sm) for sizes 568px, medium (md) at 768px, large (lg) at 1024px, and extra large (xl) at 1280px. The class names are similar to the normal grid but include the screen size right before the fraction. (.pure-u-sm-*, .pure-u-md-*, .pure-u-lg-*, and .pure-u-xl-*)

Pure Forms

Pure has three types of forms: default, stacked, and aligned. The default form is declared with the class .pure-form and is an inline version. The stacked formed helps create a nicer looking form with each element having its own row space. It is used with the class pure-form-stacked. The fanciest form is called aligned. It allows for the labels to be placed next to controls or revert on small screens. It is used with the class pure-form-aligned. It is also possible to combine grids with forms to have a multi column appearance.

Input Styles

Input sizing allows fast allocation of space for inputs. Forms are spaced using the class pure-input-* in a similar fasion to grids. Inputs can also be set to required using the attribute 'required', disabled with the attribue 'disabled' and readonly with the attribute 'readonly. Inputs can be rounded using the class pure-input-rounded.

Pure Buttons for Clicking

In the Pure framework, buttons are extremely simple to utilize. The base button class is called pure-button. Special cases can be applied to a button with one other class. Pure supports button types for disabled, active, and primary. For any other styles, Pure expects a developer to be able to create their own CSS coloring or sizing options. Buttons can be grouped in pure using the pure-button-group class.
